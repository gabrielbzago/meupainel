Política de Privacidade • TubeX

SEO Analyzer & IA para YouTube — Desenvolvido por Gabriel B. Zago

Última atualização: 27/02/2026
O TubeX somente processa dados após o consentimento explícito do usuário, exibido na primeira utilização da extensão. Nenhum dado é coletado, vendido ou utilizado para fins publicitários.

1. Informações coletadas

  • Conta Google: O e-mail pode ser utilizado temporariamente para autenticação e validação de acesso a funcionalidades, não sendo utilizado para marketing ou venda de dados.
  • Dados do YouTube: Apenas dados públicos (títulos, descrições, tags e estatísticas visíveis) são analisados, conforme permissões das APIs oficiais do Google.
  • Planos e assinaturas: O e-mail pode ser usado para validação de planos em sistemas de controle (ex: Google Sheets e Mercado Pago).

2. Uso das informações

  • As informações são utilizadas exclusivamente para fornecer funcionalidades da extensão, como análise de SEO, métricas, validação de planos e recursos com Inteligência Artificial.
  • A autenticação Google (OAuth2) é iniciada apenas quando o usuário solicita funcionalidades que exigem login.

3. Compartilhamento de dados

  • O TubeX não vende nem compartilha dados pessoais.
  • O compartilhamento ocorre apenas quando estritamente necessário para o funcionamento da extensão, utilizando: Google APIs (YouTube e OAuth2), OpenAI (IA), Google Sheets (controle de planos) e Mercado Pago (pagamentos).

3.1 Armazenamento e retenção de dados

  • Os dados processados pelo TubeX são armazenados apenas quando estritamente necessários para autenticação, validação de planos ou funcionamento das funcionalidades solicitadas pelo usuário.
  • Dados temporários utilizados para geração de conteúdo por Inteligência Artificial não são retidos após o processamento.
  • O usuário pode remover todos os dados associados ao uso da extensão a qualquer momento desinstalando o TubeX.

4. Segurança

  • Toda autenticação é realizada via OAuth2 do Google.
  • Nenhuma senha do usuário é coletada ou armazenada.

5. Direitos do usuário

  • O uso da extensão é opcional e condicionado ao aceite desta política.
  • O usuário pode remover todos os dados desinstalando a extensão.
  • Solicitações podem ser feitas pelo canal de contato abaixo.

6. Alterações nesta política

  • Esta política pode ser atualizada periodicamente.
