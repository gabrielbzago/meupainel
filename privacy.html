Política de Privacidade • TubeX

SEO Analyzer & IA para YouTube — Desenvolvido por Gabriel B. Zago

Última atualização: 25/02/2026
A extensão TubeX utiliza autenticação Google exclusivamente para identificar o usuário e liberar funcionalidades conforme o plano de acesso. A extensão analisa apenas dados públicos do YouTube e textos fornecidos pelo próprio usuário. Nenhum dado pessoal sensível é vendido ou utilizado para fins externos.

1. Informações coletadas

  • Conta Google: Apenas seu e-mail e status de inscrição no canal são consultados, nunca armazenados permanentemente em nossos servidores.
  • Dados do YouTube: Somente dados públicos de vídeos, títulos, tags e estatísticas são analisados pela extensão, conforme permissões do OAuth2 e API do YouTube.
  • Planos & Assinaturas: O seu e-mail pode ser usado para validação de planos via Google Sheets/Mercado Pago, mas nunca divulgado ou compartilhado.

2. Uso das informações

  • Acesso a planos: Seu e-mail é utilizado exclusivamente para verificar seu plano na planilha Google Sheets e liberar funcionalidades conforme o tipo de assinatura.
  • Recursos da extensão: Usamos o token OAuth2 apenas localmente. Seus dados não são transferidos para servidores de terceiros, exceto Google Sheets para controle de planos e Mercado Pago para pagamentos.

3. Compartilhamento de dados

  • Nunca vendemos ou compartilhamos seus dados pessoais.
  • Nenhum dado sensível é compartilhado com terceiros, apenas o mínimo necessário para processar pagamentos e validar o plano, via APIs oficiais Google/Mercado Pago.

Uso de Inteligência Artificial

  • O TubeX utiliza a API da OpenAI para gerar sugestões de títulos, descrições, thumbnails e análises estratégicas.
  • Os dados enviados à OpenAI consistem apenas em textos fornecidos pelo usuário ou dados públicos do YouTube.
  • Nenhuma informação pessoal sensível é enviada ou armazenada por serviços de Inteligência Artificial.

Armazenamento Local

  • Preferências do usuário, status de autenticação e plano de acesso são armazenados localmente no navegador.
  • Esses dados são removidos automaticamente quando a extensão é desinstalada.

4. Segurança

  • Toda autenticação é feita por OAuth2 do Google, garantindo máxima proteção e privacidade.
  • A API Key da extensão é usada somente para consulta pública de dados do YouTube, em conformidade com a política do Google.

5. Direitos do usuário

  • Você pode solicitar a exclusão de qualquer informação vinculada ao seu e-mail a qualquer momento, basta entrar em contato pelo suporte.
  • Para remover todos os dados, basta desinstalar a extensão. Não mantemos nenhum dado após a remoção.

6. Alterações nesta política

  • Podemos atualizar esta política ocasionalmente. Sempre publicaremos a versão mais recente neste link.
